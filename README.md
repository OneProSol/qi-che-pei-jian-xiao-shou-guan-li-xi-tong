### 一、相关技术
- 前端：`Vue`、`HTML` / `CSS` / `Javascript`。
- 后端：`Springboot`等。
- 数据库：`MySQL`

### 二、开发软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
### 三、部署所需环境

- 后端：`jdk1.8`。
- 前端：`node`、`vue-cli` 、`npm`  。
- 数据库：`mysql5.7以上`

### 四、功能描述及功能图展示
系统分为：用户、管理员。

用户功能：1. 登录、2. 注册、3. 首页、4. 公告信息、5. 配件资讯、6. 购物车、7. 个人中心、8. 我的地址、9. 我的购物车、10. 我的订单

管理员功能：1. 登录、2. 轮播图管理、3. 公告栏管理、4. 管理员管理、5. 配件资讯管理、6. 资讯分类列表管理、7. 销售系统管理、8. 分类列表管理、9. 订单列表管理、10. 员工信息管理、11. 业务员信息管理、12. 商品管理、13. 进货登记管理、14. 进货商管理、15. 销售登记管理、16. 财务统计管理
